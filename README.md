# Game-Engine-v1
My first game engine using OpenGL with now : 
- Obj file loader  
- Terrain generation using height and blend maps 
- Player and camera movement 
- Normal Mapping 
- GUI Rendering 
- Day-Night cycle skybox 
- Water Rendering using FBOs  
- Soft Shadow Mapping with PCF
